# SecureDesk DAM
Proyecto SecureDesk para el módulo de Desarrollo de Aplicaciones Multiplataforma.

## Requisitos
- XAMPP instalado
- Apache en ejecución
- PHP 8.x

## Arranque del proyecto
1. Copiar el proyecto en la carpeta `htdocs` de XAMPP.
2. Iniciar Apache desde el panel de control de XAMPP.
3. Acceder desde el navegador a:
   http://localhost/securedesk-dam/public

## Estructura del proyecto
- public/: archivos públicos y punto de entrada
- app/: lógica de la aplicación
- views/: vistas HTML
- db/: base de datos SQLite
- docs/: documentación
